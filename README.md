# Resolver Program

## Overview

Resolvers are program that is able to veto slashing incidents forwarded from networks and can be shared across networks.

## Getting started

### Build
Build the program: 
```bash
cargo-build-sbf
```

### Test
```bash
cargo nextest run --all-features
```
